# Longest Palindromic Subsctring
Given a string s, return the longest palindromic substring in s.

Example 1:
```text
Input: s = "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```
```text
Example 2:

Input: s = "cbbd"
Output: "bb"
```
```text
Example 3:

Input: s = "a"
Output: "a"
```
```text
Example 4:

Input: s = "ac"
```
